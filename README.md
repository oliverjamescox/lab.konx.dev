# Vue Base Project

This is a basic frontend only Vue project, to quickly get started with Vue & Tailwind.

This now uses Vue 3 and was built using Vue CLI.

## Development Mode

    npm run dev

Update the host url in `vue.config.js`
    
## Production Mode

    npm run build
